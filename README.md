Aplikacja konkursowa
====================

W ramach konkursu http://pretius.com/confitura2015

Wymagania
---------

* Java 8
* Maven

Budowanie i uruchamianie aplikacji
----------------------------------

```
git clone https://github.com/PanTomekPL/t-shirt-designer.git

cd t-shirt-designer

mvn spring-boot:run

open http://localhost:8080/
```
